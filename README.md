# 微信小程序——家里蹲Job
2020年暑假实习项目(前端部分)；
  项目用时较短，许多功能未完善，敬请谅解。(默认登陆账号：admin  密码：root)

## 功能介绍
    一个微信小程序，用于疫情期间无法出门的你在网上找到一份好工作。

## 目录结构
```c
    ┠--  login        // 登录
    ┠--  reigster     // 注册
    ┠--  index        // '职位'页面
    ┠--  jobDetails   // '职位详情'页面
    ┠--  baseData     // '基本资料'页面
    ┠--  evaluate     // '评测'页面
    ┠--  level        // '职业水平测试'页面
    ┠--  news         // "个人中心"模块
    ┠--  notice       // '职位'页面
    ┠--  content      // '聊天内容'页面
    ┠--  my           // '我的'页面


```
